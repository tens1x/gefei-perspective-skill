# 哥飞反模式合集（across all topics）

## 付费流量与投放

- **不要等到用户积累到一定量之后才开始考虑收费变现** — 用户一定会流失，等到数据库里有10万注册时可能已经流失了九万八千个，白白浪费掉了
  - 出处：post_05022
  - 原话：> 不要等到多少用户之后才开始考虑盈利。 用户一定会流失，很可能等到你数据库里有10万个注册用户时，已经流失了九万八千个，这部分用户就白白浪费掉了。 所以只要有人打开了产品，我们就要能够揩到一点油。
- **不要为爱发电，做产品立项之初就想好赚钱模式** — 要么收费，要么放广告，否则就是在白白浪费流量
  - 出处：post_05022
  - 原话：> 做产品，立项之初就想好赚钱模式，最好是直接向用户收费，次之是广告费。要么收费，要么放广告，千万不要为爱发电。
- **不要闭门造车，要多了解市场信息** — 技术人想创业如果不了解市场上大家怎么赚钱，容易浪费青春走弯路
  - 出处：post_00013
  - 原话：> 总结起来一句话，不要闭门造车，要多了解市场信息。
- **不要不同产品之间强行导流** — 不同产品人群不同，强行导流转化率极低，可能5800+注册只有2人付费
  - 出处：post_11552
  - 原话：> 不同产品，人群不同，不能强行导流。 我有个产品A，导流给另一个产品B，这些流量最终在B产品转化了5800+注册。 但是，但是，这些用户只有2人付费，加起来付了100美元。 人均产生LTV不到2美分。
- **不要以为ROI 100%就是回本，实体商品必须考虑利润率** — ROI 100%只是广告费等于收入，但收入里的利润可能只有20%，实际亏本80%
  - 出处：post_11291
  - 原话：> 假如ROI是100%，花掉100元广告费，卖出100元产品，得到100元收入，其中利润是20元，你会亏本80元。
- **不要新人一上手就投广告** — 广告投放需要花钱当学费，新手容易烧钱，还是先从SEO做起更稳妥
  - 出处：post_12899
  - 原话：> 所以哥飞不推荐你初期一上手就去投广告，还是先从SEO做起更稳妥。
- **不要接受来历不明的广告投放合作** — 99%是骗子，预览的广告看起来正规但链接可变，发布后会跳转到骗局页面
  - 出处：post_10935
  - 原话：> 遇到这种情况，问都不问，我会直接拉黑。  因为我知道他们99%是坏人，不敢也没必要为了那1%而去冒险。
- **不要在AppSumo上架需要支付API成本的产品** — 买断终身制且平台拿大头，有API成本的产品卖得越多亏得越多，不如直接投广告ROI可控
  - 出处：post_13090
  - 原话：> 投广告丰俭由人且ROI可控。  一旦上架到他们平台，就没有回头路可以走了。
- **不要因为一天收入很高就以为天天如此** — 偶尔的流量爆发是命运馈赠，不可控不可持续
  - 出处：post_05032
  - 原话：> 还是那句话，一天收入上万容易，一年365天天天如此就难了。
- **不要让广告转化链条过长，几个月后再回传转化数据已经没用了** — 广告引擎不知道该把广告推给谁，需要出价很高才能拿到展示机会，就会亏本
  - 出处：post_01963
  - 原话：> 后者，你几个月后再回传转化，已经没有用了。因为广告引擎不知道该把你的广告推给谁，所以需要你出价很高很高才能拿到广告展示机会，那你就亏本了。

## SEO 实战

- **不要前端渲染网页内容，要后端渲染** — 谷歌默认只抓取HTML的body内容，不会为小网站花算力执行JS代码，前端渲染导致谷歌无法收录页面内容
  - 出处：post_07478
  - 原话：> 谷歌默认抓取页面时，只抓取 html 的 body 内部内容，那么放在 script 标签里的 json 当然就不会被抓取了。  当然你会说，谷歌会去执行 js 代码。但是执行js是需要多花费算力的，谷歌凭什么给你的新网站这种待遇呢？  所以，各位请注意，如果你的网站想要从谷歌获得流量，请一定后端渲染。
- **不要为了差异化而去差异化** — 当没有经验时根本做不出差异化，用户需求摆在那里已经通过搜索词表达出来了，先做出网站获取流量再说
  - 出处：post_06198
  - 原话：> 别想着差异化。  你想再多，不如先做出一个网站，先获取到流量再说。  因为当我们还没有经验时，根本做不出差异化。
- **不要用AI生成垃圾文章做SEO** — 垃圾文章即使把访客骗进来了，也无法转化为有效顾客，谷歌只看内容质量不看生成方式
  - 出处：post_05361
  - 原话：> 垃圾文章没前途。  因为我们要靠文章吸引潜在顾客来购买服务或者购买产品，就一定不能是垃圾文章。
- **不要多语言用JS切换** — JS切换不会改变网址，导致谷歌爬虫只能抓取一种语言的内容，多语言就白做了
  - 出处：x_1783679227738653102
  - 原话：> 为什么多语言不能用js切换？ 哥飞答：因为js切换不会改变网址，这就导致谷歌爬虫来的时候，只能抓取到一种语言的内容，也就只能参与一种语言的搜索排名。那么这个网页的多语言就相当于白做了。
- **不要在meta keywords里堆砌大量无关关键字** — 搜索引擎已经降低了keywords权重，只写title和description即可
  - 出处：post_00638
  - 原话：> 后来搜索引擎排名算法就降低关键字的权重了，所以我的SEO风格就是只写title和description，不写keywords。
- **不要沉迷于免费流量而忽略放大优势** — 应该在能赚钱的地方深挖下去，不应觉得花一点时间就能做出来而不屑一顾
  - 出处：post_03469
  - 原话：> 我们还是应该放大我们的优势，哪里有流量，哪里能赚钱，就应该在哪里深挖下去。 而不应该觉得好像这是自己花一点时间就能做出来的，没啥大不了的，就不屑一顾，不重视。
- **不要使用子域名放内容，用主域名子目录** — 子目录形式的内容能把权重传递给主域名首页
  - 出处：post_00038
  - 原话：> 不要使用子域名，可以直接用主域名，子目录形式放内容，如 flomoapp.com/weekly/20211117.html 。
- **不要一个产品不成功就彻底放弃** — 应该多做几个，快速试错，快速换方向
  - 出处：post_05027
  - 原话：> 不要一个产品不成功就彻底放弃，多做几个，快速试错，快速换方向
- **不要追求完美产品再上线** — 能跑了就上线，赚到第一块钱再去优化迭代放大
  - 出处：post_05027
  - 原话：> 不要去追求完美产品，能跑了就上线，赚到了第一块钱，再去优化迭代放大
- **不要想清楚再做** — 做了才能想清楚，时间是最大的魔法
  - 出处：post_05027
  - 原话：> 不要想清楚再做，而是做了才想清楚

  - 出处：post_04344
  - 原话：> 不要说想做，以后再做。 要行动，立即开始行动。

- **不要认为做网站功能一定要多而复杂** — 只需要解决一两个核心需求，抓住精准需求做好一个功能也会有人付费使用
  - 出处：post_04905
  - 原话：> 其实很多时候我们做一个产品，并不是一定要功能多多才行，也可以只抓住一个精准需求，做好这一个功能，也会有人用，而且是付费使用。
- **不要只盯着泛流量，先搞流量再卖东西的思路不正确** — 泛流量非精准流量没意义，只有精准流量才能最高效变现

  - 原话：> 「先搞流量，然后卖什么都行」 这是一种不怎么正确的观点。  泛流量，非精准流量没啥意义。 只有精准流量才能最高效变现。
- **不要第一个网站就奔着赚大钱的目标去** — 第一个网站的目的是熟悉整套上站流程和SEO，能上线能被谷歌搜到就是成功
  - 出处：post_08023
  - 原话：> 做任何事情，都是有目的的，我们的第一个网站，目的绝对不是赚大钱。

## Adsense 与广告变现

- **不要在Adsense提示流量异常时卸载广告代码** — 卸载代码会导致系统无法持续感知你的真实流量质量，无法自动恢复广告填充率
  - 出处：post_11958
  - 原话：> 如果你的Adsense提示你流量异常，给你暂时不展示广告了，你千万不要卸载Adsense广告代码。  只要你的流量是真实的，而不是刷的，你就需要继续留着广告代码，这样系统才能持续感知你的流量质量，然后自动恢复你的广告填充率。
- **新网站不要开启自动广告** — 移动端用户容易误触自动弹窗广告，导致被谷歌判定无效流量，账号下所有网站Adsense都会被暂停
  - 出处：post_10168
  - 原话：> 实战经验，新网站先不开自动广告，防止误触导致被谷歌认为你有很多无效流量。
- **不要在网站没有流量时就去申请Adsense** — 没有流量申请一定通不过，即使能过也没有意义，应该先去搞流量
  - 出处：post_06197
  - 原话：> 刚上线，没流量，先别申请Adsense，肯定是过不了的。 再说了，即使能过，你没流量，有啥用。 所以，先去搞流量。
- **不要付费找人80美元搞100个外链** — 这种外链不仅没用还有害，会被搜索引擎惩罚
  - 出处：post_08077
  - 原话：> 做外链有很多方法，但是付费找人80美元搞100个外链这种就不要去做了，这种外链不仅没用还有害。
- **不要看不上小钱，只想着找能赚大钱的生意** — 时间会浪费在寻找上，赚钱靠经验积累，只有赚过很多小钱才能领悟赚更多钱的诀窍
  - 出处：post_05801
  - 原话：> 如果看不上小钱，一心只想着找能赚大钱的生意，那么时间就会浪费在找的过程中。
- **不要觉得做网站赚广告费不是正道而放弃** — 哥飞自己就因为觉得不是正道而错失了持续深耕的机会，应该放大优势在能赚钱的地方深挖
  - 出处：post_03469
  - 原话：> 但是，可惜，没坚持。 当时总觉得这东西不是正道，没有花更多精力下去，上了几个站点后就放在那里，没去管它。
- **不要想着靠AI自主生成内容就能躺赚** — 谷歌能检测到伪原创和胡编乱造的内容，打上低信任度标签拿不到排名，之前已经K过不少内容农场
  - 出处：post_08077
  - 原话：> 更别说所谓的伪原创技术，没用的，现在AI很强了，这些都能够检测出来。 要做内容就老老实实做内容，磨刀不误砍柴工。谷歌之前可是K过不少内容农场的，现在想仅仅靠AI自主生成内容就躺赚是不可能的了。
- **不要随便搞一个页面就去糊弄Adsense审核** — 网站要看起来完善完整，不能随便弄一个糊弄审核的页面，否则会被搞黑灰产的人波及

  - 原话：> 加上，是让网站看起来更完善，更像是一个完整的网站，而不是随便弄一个糊弄审核的页面。

  - 出处：post_03913
  - 原话：> 我年轻时不懂，曾经因为帮同学收过几万美元他香港公司的美元，后来自己收 Adsense 的电汇时就遇到过这个提示。 只能等几年后限制自动失效了。
- **不要做SEO太心急，不能想着立即就要拿到结果** — 做SEO要跟时间交朋友，有些网站需要好几个月才能起来，耐心坚持才会有收获

## AI 工具与效率

- **不要乱填你的 API Key，尤其是有特殊权限的号** — 别人得到你的 API key 后可以盗你的号，OpenAI 账号有漏洞
  - 出处：post_04459
  - 原话：> 所以不要乱填你的 key ，尤其是你有特殊权限，如Plugin 开发者权限、GPT4API 权限的号。 也不要同意别人说借用你的 api key 做测试。
- **不要用 AI 做事时放弃自己擅长的方式，去试用不熟悉的新工具** — 会浪费大量时间，最终还得回到自己擅长的方式
  - 出处：zhihu_p1952345403925192806
  - 原话：> 所以，总结一下，这次写 PPT 浪费时间，主要原因是放弃了自己擅长的做事方式，试图用一个新工具来实现。  这就告诉我们，做事情，还是要先用自己熟悉的擅长的方式去做。
- **不要让 Cursor 全自动驾驶写复杂项目的核心逻辑** — AI 生成大量代码后你不了解代码，遇到小问题容易越改越错，之前正常的逻辑也可能被破坏
  - 出处：post_11178
  - 原话：> 同感，核心逻辑还是要自己写，再让 Cursor 打下手，会更好
- **不要只看 AI 生成的精美总结图片就以为自己理解了** — 大量的实操经验隐藏在细节里，图片无法展示细节
  - 出处：post_12908
  - 原话：> 图片看起来的确很精美，也会有人收藏。 但如果只看图片，是无法了解细节的。 而大量的实操经验，又隐藏在细节里。 所以，你看到别人传播这类看起来精美的图片，就要问一问自己，真的理解了吗？
- **不要因为不完美就不推出产品** — 产品永远有值得改进的点，先占住生态位再不断迭代才是正确策略
  - 出处：post_12645
  - 原话：> 有些人开发产品的心态是，我不打磨到100分，我就不推出来。  然而，其实产品永远有值得改进的点，难到那就一直改进，一直不推出吗？
- **不要利用别人免费开放的接口批量拉数据再转卖** — 会导致接口被限制，大家都没得用，而且无法兑现已收费的承诺
  - 出处：post_14230
  - 原话：> 本来这就是别人故意露出来的一个免费接口，大家自己用用得了。 就有人要去利用这个接口批量拉数据，还把数据包装一下卖给同行。  现在，这个接口开始限制访问了
- **不要为难自己一上手就去学做 App** — 网页特别容易被传播和使用，不需要审核，部署在免费基础设施上成本极低
  - 出处：post_13134
  - 原话：> 你没必要为难自己，一上手就去学App是不明智的，特别轻量级的网页绝对是你的最佳选择。
- **不要相信 AI 说自己能联网的话** — AI 会根据你的问题迎合你来回答，实际并没有联网能力
  - 出处：post_05603
  - 原话：> 我记得看到有即友说 Claude2 可以联网，然后我实际测试发现，根本不能联网，就是胡说八道。 为什么说是胡说八道呢？看图二就知道，他是根据你的问题迎合你来回答的。 所以结论就是，Claude2根本就没联网。
- **不要对不懂的领域随便下结论** — 大部分人只会用老经验老眼光看待新事物，自以为看懂了其实没看懂

## 定价与客单价

- **不要定价低于两位数，不要不好意思收钱** — 贱卖自己的创意和劳动力，低价带来更多客诉退款和更少收入
  - 出处：post_04457
  - 原话：> 定价不要低于两位数 不要贱卖自己的创意和劳动力，如果你连让用户掏一百块的信心都没有，建议换个产品。
- **不要想着先积累用户再收费** — 流量进入产品就要转化产生收入，否则白白浪费流量
  - 出处：post_04457
  - 原话：> 上线第一天就要收费 不要想着先积累用户，不要靠广告，流量进入产品就要转化，产生收入。
- **不要打折降低客单价** — 打折会拉低客单价，不如加量（如买一年送2月）来维持价格体系
  - 出处：post_04457
  - 原话：> 营销打折不如加量 不要打折降低客单价，可以买一年送 2 月。
- **不要为了微小的单价优惠购买过大的会过期套餐包** — 用不完就浪费了，实际单价并没有便宜多少
  - 出处：post_02384
  - 原话：> 买10万条的就足够了，没必要为了省那1分钱而买50万条的。
- **不要承诺终身会员制** — 承诺终身制就相当于没交付，谁也不敢保证未来能持续服务
  - 出处：post_12938
  - 原话：> 是的，这里不能免费，不能承诺终身制。 承诺终身制，其实就相当于没交付。

  - 出处：post_13526

- **不要以为用户量越大才越赚钱** — 赚钱要看客单价和流量精准度，而不是单看用户量
  - 出处：post_09458
  - 原话：> 典型的没做过赚钱产品的人的思维逻辑，他以为用户量越大才越赚钱。 实际要看客单价的，也要看流量是否精准。
- **不要扭扭捏捏定个$4.9或$6.9的低价** — 虽然可以提高付费率，但无法收益最大化
  - 出处：post_14571
  - 原话：> 敢定高价的意思是，不要不好意思要钱，你扭扭捏捏的定个$4.9或者$6.9的确能够提高付费率，但无法收益最大化。
- **不要直接全抄别人的定价和文案** — 不动脑子照搬别人的成果是不道德的，也不会有好结果
  - 出处：post_14230
  - 原话：> 结果这个大聪明回去后，就做了个 AI 工具站， 逮着我们某位也在即刻的朋友的网站从文案到定价直接全抄，一点脑子都不动的。
- **不要搞邀请返佣机制** — 分销佣金让推荐动机不纯，应该让推荐出于真心认可
